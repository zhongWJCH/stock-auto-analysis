基于你提出的“仅从面上数据”这一前提，下面是一些**纯粹基于公开数据的量化策略**。

这些策略的核心是利用历史规律，将“不确定性”转化为“概率优势”。我会从简单到复杂，给你三个层次的策略思路，你可以根据自己的技术偏好来选择。

---

### 策略一：基于“超跌反弹”的均值回归策略

**核心逻辑**：短期内价格跌得过多，存在向均值回归的动力。

- **选股条件**（同时满足）：
    1.  **RSI(14) < 30**：指标进入超卖区，短期下跌动能衰竭。
    2.  **股价偏离20日均线 > -10%**：说明短期偏离度较大，有反弹需求。
    3.  **成交量萎缩**：当日成交量低于过去5日均量的50%以上，表明抛压枯竭。

- **操作时机**：当日收盘符合以上条件，次日开盘价买入。

- **卖出时机**：
    1.  当股价反弹至**10日均线**附近。
    2.  或持仓**5个交易日**，无论盈亏都平仓。
    3.  或**跌破买入价5%** 止损。

- **备注**：该策略在震荡市或下跌末期效果好，在单边下跌的强趋势中容易“抄底抄在半山腰”。

---

### 策略二：基于“强者恒强”的动量突破策略

**核心逻辑**：一只股票如果能以放量长阳突破关键阻力位，后市大概率继续上涨。

- **选股条件**（同时满足）：
    1.  **放量突破**：当日涨幅 > 5%，且成交量是过去**20日均量**的**1.5倍**以上。
    2.  **突破平台**：收盘价创**过去20个交易日**的最高价。
    3.  **相对强势**：当日涨幅跑赢大盘（如上证指数）2个百分点以上。

- **操作时机**：当日收盘符合条件，次日开盘价买入。

- **卖出时机**：
    1.  买入后，**3日内**股价未能继续创出新高。
    2.  股价跌破**5日均线**时卖出。
    3.  严格执行**-5%** 的硬止损。

- **备注**：这是典型的动量交易，在强势牛市或主流板块中效果显著。但在震荡市中容易被反复“假突破”欺骗。

### 策略三：基于“量价行为”的综合评分策略

**核心逻辑**：将多个独立指标组合成一个评分系统，寻找“共振”最强的个股。

**评分模型（总分10分）**：

| 维度 | 子条件 | 得分 |
|------|--------|------|
| **趋势** | 收盘价站上**20日均线** | +2分 |
| | **20日均线**方向向上 | +1分 |
| **动量** | MACD指标在零轴上方，且DIFF线向上 | +2分 |
| **量能** | 成交量 > 过去**5日均量** | +1分 |
| | 当日为**阳线**（收盘价 > 开盘价） | +1分 |
| **资金** | 资金流向（如DDE大单净量）为正 | +2分 |
| **情绪** | 所属板块当日涨幅排名市场前20% | +1分 |

- **操作策略**：
    -   **买入信号**：选择综合得分 > 7分的股票，在次日开盘买入。
    -   **卖出信号**：
        1.  持仓股票的综合得分降至 **4分以下**。
        2.  股价跌破**10日均线**。
        3.  设置**-5%** 的止损线。

- **备注**：这个策略可以通过自己设定权重来调整，属于可以自己编写的低配量化模型。
