# 场内 ETF 策略分析网站

这是一个 `Vite + Vue 3 + Element Plus` 的本地网站，用来完成以下目标：

- 展示场内 ETF 的本地化行情信息
- 提供数据下载与更新操作
- 在侧边展示美元/人民币、黄金、石油、美元指数
- 按 `策略.md` 的五套 ETF 策略对历史日期做预测
- 按 `验证.md` 输出胜率、盈亏比、最大回撤
- 支持多股对比与历史真实结果回看

## 启动方式

1. 联网同步真实数据：

```bash
npm run sync -- --start=2024-01-01
```

2. 如果当前环境暂时不能联网，先写入演示数据：

```bash
npm run seed
```

3. 前端开发模式：

```bash
npm run dev
```

说明：`npm run dev` 之前会自动执行一次同步脚本，把数据写到 `public/data/`，页面读取这些静态 JSON，并在前端完成策略计算。

4. 构建并预览网站：

```bash
npm run build
npm run preview
```

打开 `http://localhost:5173` 或预览地址

## 当前实现说明

- 主数据源：东方财富公开接口
- 数据同步：默认同步场内 ETF，并额外同步沪深300、中证全指和上证指数作为基准
- 历史数据：同步脚本会把历史和快照写到 `public/data/`
- 侧边行情是快照值，仅用于展示，不做实时交易决策
- 相对强弱基准：优先使用 `沪深300(000300)`，其次 `中证全指(000985)`；缺失时使用 `上证指数(000001)`
- 策略预测：以所选日期收盘为信号生成时点，预测下一交易日表现，并用真实后续走势验证
- 运行方式：只有下载脚本保留，页面不再依赖独立 Node API 服务
- 不实现实时套利、折溢价套利和高频网格，因为本项目只使用本地日线数据，无法可靠处理盘口、申赎或实时溢价

## 目录结构

- `src/ui/`：Vue + Element Plus 前端
- `src/data-sources/eastmoney.js`：数据源适配
- `src/services/app-service.js`：下载脚本入口
- `src/services/strategy-service.js`：策略与验证逻辑
